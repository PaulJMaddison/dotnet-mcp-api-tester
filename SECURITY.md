# Security Policy

## Supported Versions

Security fixes are prioritized for the latest release on `main`.

## Reporting a Vulnerability

If you discover a security issue, please report it privately:

- Email: **security@apitester.dev**
- Include: affected component, impact, proof-of-concept or reproduction steps, and suggested mitigations if available.

### Disclosure Process

1. We acknowledge receipt within **3 business days**.
2. We triage and validate the issue, then share an initial severity assessment.
3. We coordinate remediation and a disclosure timeline with the reporter.
4. We publish a security advisory after a fix is available, or after agreed disclosure timelines are met.

Please do not open public issues for suspected vulnerabilities.
